1. Kurulum — API Ayarları
Google Client ID nedir?
Uygulamanın Google hesabına erişmesi için gerekli tanımlayıcı.
  1. console.cloud.google.com adresine git
  2. Soldaki menüden APIs & Services → Credentials seç
  3. Üstte + CREATE CREDENTIALS → OAuth client ID tıkla
  4. İlk kez yapıyorsan Configure Consent Screen çıkar:
    → User Type: External → Create
    → App name, e-posta gir → kaydet, geri kalanı atla
  5. Application type: Web application seç
  6. Authorized redirect URIs alanına şu adresi ekle:
  7. CREATE tıkla — açılan penceredeki Client ID değerini kopyala
Google Client Secret nedir?
Client ID ile birlikte gelen gizli anahtar; Google'ın OAuth akışında token alışverişi için zorunlu.
  1. Yukarıdaki adımlarla aynı sayfada (Credentials) Client ID'ni oluşturduktan sonra açılan popup'ta hem ID hem de Client secret görünür
  2. Popup'ı kapattıysan: listedeki Client ID'ye tıkla → "Client secret" alanını bul → gözün simgesine basarak göster ve kopyala
🔒 Güvenlik notu: Bu değer yalnızca tarayıcının yerel deposunda (localStorage) saklanır. Hiçbir sunucuya gönderilmez.
Gemini API Key nedir?
Yapay zeka analizleri için gerekli; ücretsiz kullanım limiti günlük 1500 istek.
  1. aistudio.google.com/apikey adresine git
  2. Get API Key → Create API key tıkla
  3. Yeni proje seç ya da mevcut projeyi kullan
  4. Oluşan AIza... ile başlayan anahtarı kopyala
💡 Ücretsiz mi? Evet. Kart bilgisi girmeden, günde 1500 istek sınırıyla ücretsiz kullanılabilir. Sınav çalışması için bu limit fazlasıyla yeterli.

Tüm anahtarlar yalnızca tarayıcında saklanır, hiçbir sunucuya gönderilmez.

Gemini Modeli

Google zaman zaman yeni Gemini modelleri çıkarır veya eski modelleri kaldırır. Bir model artık çalışmıyorsa (örn. "model bulunamadı" hatası alıyorsan), kodu değiştirmene gerek yok — güncel model adını buraya yazman yeterli.

Geçerli model adlarını Google'ın model listesinden kontrol edebilirsin. Boş bırakırsan varsayılan () kullanılır.
